# TORQ Project

This README provides details for the TORQ project, including setup instructions, usage guides, and various components.

## Project Structure

- **backend/**: Contains the backend API setup.
- **frontend/**: Contains the frontend dashboard.
- **database/**: Contains the database schema and migrations.
- **scrapers/**: Contains web scrapers for data collection.
- **docs/**: Documentation and guides.

## Setup Instructions

### Backend
1. Navigate to the `backend` directory.
2. Install dependencies.
3. Start the server.

### Frontend
1. Navigate to the `frontend` directory.
2. Install frontend dependencies.
3. Start the frontend application.

## Usage

- Access the API at `http://localhost:5000/api`
- Access the frontend at `http://localhost:3000`
